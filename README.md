Project Title:
        Call Center Performance Dashboard

Project Overview:
        An interactive Power BI dashboard designed to track and analyze call center operations. It provides a comprehensive view of agent productivity, service efficiency, and customer satisfaction levels to drive data-informed management decisions.

Key Metrics / Insights:
        Call Volume Analysis: Monitoring total calls, answered vs. abandoned rates, and monthly trends.
        Service Quality: Tracking average speed of answer and issue resolution rates.
        Customer Experience: Measuring average satisfaction ratings through interactive gauge visuals.
        Agent Performance: Detailed breakdown of individual metrics including calls handled, resolution speed, and customer feedback.

Tools & Technologies:
        Power BI Desktop: For end-to-end dashboard development and visualization.
        Power Query: For data cleaning, transformation, and modeling.
        DAX (Data Analysis Expressions): For creating advanced measures.
        Interactive Visuals: Utilization of Gauges, Donut Charts, and Slicers for dynamic data exploration.
